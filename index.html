Simon Game
Game information

Objective

When you press the 'Start' button, the game will begin with a size increase of 1 of the 4 coloured pads from the board and will increment by 1 on the number display with each pattern you've followed correctly. Your goal is to repeat the pattern that has been generated by the game as long as you can and make a bigger record. This game is really good for training your memory skills so stay focussed and be carefull not to make a mistake! When you reach 25 the game is considerred as won!

Rules

  • Game can be won only while 'strict' mode is active.
  • If 'Strict' mode is not active and you click to activate it, the number will go back to zero and the game will generated a new sequence.
  • You could switch off 'Strict' mode at any point
  • You can start a new game at any given time by pressing the 'Start' button.

Buttons

  • Start - To begin the game, press 'Start' and your first sequence will be generated by the game. You can do this at any point.
  • Strict - When 'Strict' mode is activated (blue), if you get your sequence wrong the game will be over. Although if its not activated (white) you can try and repeat the sequence as many times as you like, until you get it correct and move to the next sequence.
